# Log Regex Web Tool

Node.js веб-приложение для управления файлами на сервере и обработки логов регулярными выражениями.

## Возможности

- Дерево директорий `./root` с операциями:
  - загрузка файла,
  - создание папки,
  - удаление,
  - переименование,
  - скачивание файла,
  - перемещение drag&drop.
- Правая панель для обработки логов:
  - ввод входной директории (`/logs` => `./root/logs`),
  - ввод выходного файла (`/result.txt` => `./root/result.txt`),
  - позитивный/негативный regex-фильтр,
  - список extractor regex (каждый должен содержать 1 группу),
  - история regex + подсказки через `datalist` и `localStorage`,
  - шаблон итоговой строки с плейсхолдерами `{name}`.

## Запуск через Docker Compose

```bash
docker compose up --build
```

Приложение будет доступно на `http://localhost:3000`.

Папка `./root` на хосте примонтирована в контейнер как `/data/root`, поэтому вы можете ассоциировать свой путь к данным через volume (измените `docker-compose.yml` при необходимости).

## Локальный запуск

```bash
npm install
npm start
```

Переменная `ROOT_DIR` задаёт путь к корневой директории для файлов.
