# Log Regex Workbench

Web-приложение на Node.js для управления файловым деревом и обработки логов regex-шаблонами.

## Возможности

- Файловое дерево в `./root`: загрузка, удаление, переименование, скачивание, перемещение drag-and-drop, создание папок.
- Форма обработки:
  - директория-источник (`/logs` -> `./root/logs`),
  - выходной файл (`/result.txt`),
  - include/exclude regex-фильтр,
  - несколько regex-правил с одной группой,
  - история regex по именам полей,
  - шаблон финальной строки с плейсхолдерами (`{ip}`, `{site}`).

## Запуск локально

```bash
npm install
npm start
```

Приложение откроется на `http://localhost:3000`.

## Запуск в Docker

```bash
docker compose up --build
```

Корень файловой системы внутри приложения мапится из `./root` через volume в `docker-compose.yml`.
