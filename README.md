# Log Processor Web

Web-приложение на Node.js для управления деревом файлов в `./root` и обработки логов по регулярным выражениям.

## Возможности

- Дерево файлов/папок с операциями:
  - загрузка файлов;
  - создание папок;
  - переименование;
  - удаление;
  - перемещение drag-and-drop;
  - скачивание файлов.
- Интерфейс обработки логов:
  - входная директория (например `/logs`);
  - выходной файл (например `/result.txt`);
  - фильтр regex (позитивный или негативный);
  - список regex-паттернов с ключами (`ip`, `site` и т.д.), каждый regex должен иметь ровно 1 группу;
  - шаблон выходной строки с плейсхолдерами вида `{ip}`.
- История regex (через `localStorage`) для подсказок в полях паттернов.

## Локальный запуск

```bash
npm install
npm start
```

Открыть: `http://localhost:3000`.

## Запуск в Docker

```bash
docker compose up --build
```

По умолчанию `./root` на хосте монтируется в контейнер как `/data/root`.

Если нужно, можно переназначить volume в `docker-compose.yml`.
